# GreDDBot 

GreDDBot is a twitch chat bot created by [GreDDySS](https://twitch.tv/greddyss)

---

A little guide on how to get started with the bot.

1. You need to rename the .env.example file to .env or create a new .env file.

2. Fill in your data

3. Create tables in sql. Scheme in [sql-scheme](/sql-sheme.md)

4. Make sure to install **npm** with these Commands:
```
npm i
npm start
```

### Done
Your bot is ready to work.
